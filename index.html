Tipografija u Najboljem Svjetlu

Ovo je primjer jednostavne web stranice kreirane koristeći osnovne HTML oznake i CSS stilove. Obavljeni zadatak predstavlja primjenu osnosvnih html. i css. tagova kao što su: html, head, style/link, body, h1 i p.

U ovoj vježbi koristimo dva web fonta sa stranice Google Fonts, "Roboto" i "Playfair Display", te prilagođavamo izgled naslova i paragrafa putem CSS svojstava za boje, pozadinu i fontove.
